# AlgorithmReview

## กรณีศึกษา 1 : Array
- คำนวณคะแนนรวม
- คะแนนเฉลี่ย
- คะแนนสูงสุด
- คะแนนต่ำสุด
- จำนวนผู้ได้คะแนนตั้งแต่ 7 ขึ้นไป
- รายชื่อนักศึกษาที่ได้คะแนนต่ำกว่า 5

## กรณีศึกษา 2 : Stack
- เพิ่มข้อมูลลง Stack
- Undo 2 ครั้ง
- แสดงข้อมูลหลัง Undo
- อธิบายหลักการ LIFO

## กรณีศึกษา 3 : Queue
- เพิ่มผู้ป่วยเข้าคิว
- เรียกผู้ป่วยออก 2 คน
- เพิ่มผู้ป่วยใหม่
- แสดงผู้ป่วยคนถัดไป
- แสดงจำนวนผู้ป่วย
- อธิบายหลักการ FIFO
